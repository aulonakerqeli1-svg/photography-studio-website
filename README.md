## Features

- **Home Section:** Hero banner with tagline and call-to-action button.
- **About Section:** Studio background and achievements.
- **Gallery Section:** Horizontal scrolling image gallery showcasing work.
- **Services Section:** Highlighting different photography services offered.
- **Pricing Section:** Various packages for clients to choose from.
- **Contact Section:** Contact form, address, email, phone, and Google Maps integration.
- **Responsive Design:** Optimized for mobile, tablet, and desktop devices.
- **Dark Theme:** Modern dark color palette with accent highlights.
